# ❓ سؤالات متداول (FAQ) - Inventory Sync Pro

## سؤالات نصب و راه‌اندازی

### ❓ پلاگین نصب است اما منو ظاهر نمی‌شود
**جواب:**
```
1. برای Inventory Sync Admin بودن ضروری است
2. صفحه را جدید بارگذاری کنید (Ctrl+Shift+Del)
3. اگر هنوز نشد، پلاگین را غیرفعال و دوباره فعال کنید
4. WordPress > تنظیمات > General > Permalink را SAVE کنید
```

### ❓ چگونه API Credentials را پیدا کنم؟
**جواب:**
```
1. WooCommerce > Settings > Advanced > REST API
2. Create an API credential
3. Permissions: Read and Write انتخاب کنید
4. Consumer Key و Secret را کپی کنید
```

### ❓ آیا می‌توانم بیش از 2 سایت را متصل کنم؟
**جواب:**
```
الان فقط 2 سایت پشتیبانی می‌شود
اما می‌توانید 2 سایت A و B را متصل کنید
و پس از آن B و C را جداگانه متصل کنید
```

### ❓ آیا قدیمی‌ترین نسخه‌های WooCommerce پشتیبانی می‌شود؟
**جواب:**
```
نسخه حداقل: WooCommerce 4.0
نسخه جدید: WooCommerce 8.0+
اگر نسخه قدیمی دارید، بروزرسانی کنید
```

## سؤالات تنظیمات

### ❓ بهتر است جهت هماهنگ‌سازی کدام باشد؟
**جواب:**
```
سایت 1 → سایت 2:
└─ سایت 1 مرجع است، سایت 2 دنبال می‌کند

سایت 2 → سایت 1:
└─ سایت 2 مرجع است، سایت 1 دنبال می‌کند

دوطرفه:
└─ هر دو سایت دارای موجودی یکسان
└─ کند‌تر است، محتاط باشید
```

### ❓ هماهنگ‌سازی خودکار چقدر مشکل ساز است؟
**جواب:**
```
هماهنگ‌سازی هر 5 دقیقه اتفاق می‌افتد
└─ تأثیری بر سرعت سایت ندارد
└─ از Cron کار دشوار استفاده می‌کند
└─ می‌توانید بندصورت تنظیم کنید
```

### ❓ آیا قیمت‌ها هماهنگ می‌شوند؟
**جواب:**
```
نه! فقط موجودی و محصولات هماهنگ می‌شوند
قیمت‌ها مستقل هستند
└─ هر سایت قیمت خود را دارد
```

## سؤالات انتقال محصولات

### ❓ اگر محصولات قبلاً وجود دارند چه کنم؟
**جواب:**
```
بدون اینکه دوبار منتقل شوند:
1. به تب "مرتبط‌سازی محصولات" بروید
2. محصولات را انتخاب کنید
3. "ذخیره نقشه‌برداری" را کلیک کنید
└─ اکنون مرتبط شده‌اند
```

### ❓ عکس‌ها منتقل می‌شوند؟
**جواب:**
```
بله! تمام عکس‌ها منتقل می‌شوند
└─ عکس‌های معمولی ✅
└─ عکس‌های متغیر ✅
└─ عکس‌های دسته‌بندی ✅
```

### ❓ محصولات به‌صورت published یا draft منتقل می‌شوند؟
**جواب:**
```
به‌صورت DRAFT منتقل می‌شوند!
└─ شما باید آن‌ها را بررسی کنید
└─ سپس publish کنید
└─ این برای ایمنی است
```

### ❓ اگر محصول منتقل نشد چی؟
**جواب:**
```
1. لاگ‌ها را در تب لاگ‌ها بررسی کنید
2. معنی پیام خطا را متوجه شوید
3. مشکل را حل کنید
4. دوباره انتقال دهید
```

## سؤالات موجودی

### ❓ موجودی کی بروزرسانی می‌شود؟
**جواب:**
```

└─ وقتی شما "هماهنگ‌سازی" را کلیک کنید

└─ فوری (اگر خودکار فعال است)
└─ یا بعد از 5 دقیقه
```

**جواب:**
```

└─ سایت 1: 100
└─ سایت 2: 50
└─ AFTER SYNC: هر دو 100

سایت 2 → سایت 1:
└─ AFTER SYNC: هر دو 50

دوطرفه:
└─ مقدار بالاتر انتخاب می‌شود
```

### ❓ آیا موجودی منفی ممکن است؟
**جواب:**
```
نه! موجودی منفی نمی‌شود
└─ حداقل مقدار: 0
```

**جواب:**

```
└─ بهتر است شب انجام دهید
```


## سؤالات خرابی‌ها

### ❓ "API Error" چی به معنی است؟
**جواب:**
```
API دسترسی ندارد
حل:
1. API تفعیل است؟
2. API Key صحیح است؟
3. Permissions Read/Write است؟
4. سایت آنلاین است؟
```

### ❓ "Connection Timeout" چی است؟
**جواب:**
```
سایت خیلی کند پاسخ می‌دهد
حل:
1. سایت خراب نیست؟
2. Internet سریع است؟
3. سایت بسته نیست؟
4. دوباره تلاش کنید
```

### ❓ موجودی‌ها معکوس هستند (0 جای 100)
**جواب:**
```
چیزی اشتباه است!
حل:
1. لاگ‌ها را بررسی کنید
2. تب تنظیمات بررسی کنید
3. جهت هماهنگ‌سازی درست است؟
4. دوبار sync کنید
```

## سؤالات پیشرفته

### ❓ آیا می‌توانم Cron Interval را تغییر دهم؟
**جواب:**
```
فعلاً از داشبورد نمی‌توانید
اما می‌تواند از کد تغییر یابد:

wp_schedule_event(
    time(), 
    'interval_name',  // تغییر دهید
    'inventory_sync_cron_hook'
);
```

### ❓ آیا WordPress Cron تنظیم شده است؟
**جواب:**
```
بررسی کنید:
1. wp-config.php را باز کنید
2. اگر این سطر وجود ندارد، اضافه کنید:
   define('DISABLE_WP_CRON', false);
3. Save کنید
```

### ❓ آیا می‌تونم API را Custom کنم؟
**جواب:**
```
بله! Hooks استفاده کنید:

do_action('inventory_sync_before_sync');
do_action('inventory_sync_after_sync');

apply_filters('inventory_sync_verify_ssl');
```
