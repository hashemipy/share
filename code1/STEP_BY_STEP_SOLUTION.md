
## مرحله ۲: فایل JSON را صحیح کنید

### فایلی که دانلود کردید معیوب است!

```json
{
  "products": [
    {
      "id": 123,
      "name": "محصول متغیّر",
      "type": "variable",     // ✅
      "attributes": [         // ✅
        {
          "id": 1,
          "name": "رنگ",
          "options": ["قرمز", "آبی"]
        }
      ],
      "variations": [         // ✅ شامل متغیّرها
        {
          "sku": "var-1",
          "attributes": [
            {
              "id": 1,
              "name": "رنگ",
              "option": "قرمز"
            }
          ]
        }
      ]
    }
  ]
}
```

## مرحله ۳: آپلود کنید

1. وارد WordPress شوید
2. **WooCommerce → دانلود/آپلود محصول**
3. فایل JSON را انتخاب کنید
4. **آپلود** را کلیک کنید

---

## مرحله ۴: بررسی کنید

وارد محصول شوید و چک کنید:

```
✅ نوع: متغیّر
✅ ویژگی‌های محصول: مثلاً "رنگ"
✅ تعداد متغیّرات: مثلاً 2 یا بیشتر
✅ تصاویر: دانلود شده
```

---

## 🔧 اگر هنوز کار نکرد:

### مشکل: محصول ساده ایجاد شد

**دلیل:** فایل JSON شامل `variations` نیست یا خالی است

**حل:**
```json
// ❌ اشتباه
{
  "type": "variable",
  "attributes": [...],
  "variations": []
}

// ✅ درست
{
  "type": "variable",
  "attributes": [...],
  "variations": [
    { "sku": "var-1", "attributes": [...] }
  ]
}
```

### مشکل: ویژگی‌ها نمایش داده نمی‌شوند

**دلیل:** `attributes` بدون `options` است

**حل:**
```json
// ❌ اشتباه
{
  "attributes": [
    { "id": 1, "name": "رنگ" }
  ]
}

// ✅ درست
{
  "attributes": [
    { "id": 1, "name": "رنگ", "options": ["قرمز", "آبی"] }
  ]
}
```

### مشکل: متغیّرات ایجاد نشدند

**دلیل:** `variations` بدون `attributes` است

**حل:**
```json
// ❌ اشتباه
{
  "variations": [
    { "sku": "var-1", "price": "150000" }
  ]
}

// ✅ درست
{
  "variations": [
    {
      "sku": "var-1",
      "price": "150000",
      "attributes": [
        { "id": 1, "name": "رنگ", "option": "قرمز" }
      ]
    }
  ]
}
```

### مشکل: تصاویر دانلود نشدند

**دلیل:** تصویر URL معتبر نیست یا دسترسی ندارید

**حل:**
```json
// ✅ تصویر معتبر
{
  "images": [
    {
      "id": 1,
      "src": "https://example.com/valid-image.jpg"
    }
  ]
}
```
