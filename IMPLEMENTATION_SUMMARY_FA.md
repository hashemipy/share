## فلوی کار نهایی:

1. کاربر وارد تب Settings می‌شود
2. انتخاب می‌کند: "سایت 1" یا "سایت 2"
3. ذخیره تنظیمات (صفحه تحدیث می‌شود)
4. اگر "سایت 1" انتخاب شد، تب "مرتبط‌سازی محصولات" فعال می‌شود
5. کاربر یک محصول از سایت 1 انتخاب می‌کند
6. یک محصول از سایت 2 انتخاب می‌کند
7. دکمه "ایجاد ارتباط" را کلیک می‌کند
8. نقشه‌برداری در دیتابیس ذخیره می‌شود
9. از آن پس، هر تغییر در موجودی این دو محصول هماهنگ می‌شود (خودکار)

===========================================

## نکات مهم ایمنی:

❌ Direct Database Queries: استفاده نمی‌شود
✅ Prepared Statements: برای تمام Query ها
✅ Hooks: استفاده می‌شود برای هماهنگ‌سازی خودکار
✅ Nonce Verification: برای تمام AJAX requests
✅ Permission Checking: فقط کاربران manage_woocommerce می‌توانند دسترسی داشته باشند

## راه‌های آزمایش:

1. به WordPress داشبورد بروید
2. به Inventory Sync > تنظیمات برگردید
3. انتخاب کنید: "سایت 1" یا "سایت 2"
4. دکمه "ذخیره تنظیمات" کلیک کنید
5. اگر "سایت 1" انتخاب کردید:
   - تب "مرتبط‌سازی محصولات" دیده می‌شود
   - جستجو برای محصولات کار می‌کند
   - انتخاب دو محصول و کلیک "ایجاد ارتباط" نقشه‌برداری ایجاد می‌کند

اگر "سایت 2" انتخاب کردید:
   - تب "مرتبط‌سازی محصولات" پنهان است
   - پیام: "این ویژگی فقط در سایت 1 دسترسی‌پذیر است"
