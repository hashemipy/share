## 🔧 نصب در 3 مرحله:

### **مرحله 1: کپی پوشه‌ها**

**سایت 1 (Master):**
```
پوشه: inventory-sync-master
مقصد: /wp-content/plugins/inventory-sync-master/
```

**سایت 2 (Slave):**
```
پوشه: inventory-sync-slave
مقصد: /wp-content/plugins/inventory-sync-slave/
```

استفاده از SFTP یا File Manager هوستینگ:
1. پوشه `inventory-sync-master` را دانلود کنید
2. به FTP متصل شوید
3. پوشه را در `/wp-content/plugins/` آپلود کنید

### **مرحله 2: فعال‌سازی**

**در هر دو سایت:**
1. وارد داشبورد WordPress شوید
2. به "Plugins" بروید
3. پلاگین را فعال کنید

### **مرحله 3: تأیید نصب**

**سایت 1:**
- منو "هماهنگ‌سازی انبار" ظاهر شد؟ ✅
