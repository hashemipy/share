# تاریخچه تغییرات - Inventory Sync Pro

## نسخه 2.0.0 - بهبود انتقال محصولات

### ویژگی‌های جدید:

#### 1. **انتقال خودکار دسته‌بندی‌ها و ویژگی‌ها** ✨
- هنگام انتقال محصول، تمام دسته‌بندی‌های والد و فرزند خودکار منتقل می‌شوند
- اگر دسته‌بندی در سایت 2 موجود بود، استفاده می‌شود؛ در غیر اینصورت ایجاد می‌شود
- تمام ویژگی‌های محصول (رنگ، سایز، و غیره) و تکیه‌های آن منتقل می‌شوند
- اگر ویژگی درحالِ انتقال با ویژگی موجود در سایت 2 تطابق داشته باشد، استفاده می‌شود

#### 2. **کلاس جدید Category Attribute Sync** 🔄
فایل: `includes/class-category-attribute-sync.php`
- `sync_product_categories()` - انتقال دسته‌بندی‌های محصول
- `sync_product_attributes()` - انتقال ویژگی‌های محصول
- `sync_attribute_terms()` - انتقال تکیه‌های ویژگی (مثل: رنگ قرمز، سایز بزرگ)
- جستجوی هوشمند برای دسته‌بندی و ویژگی موجود

#### 3. **جدول‌های دیتابیسی جدید** 📊
- `inventory_sync_category_mapping` - نقشه‌برداری دسته‌بندی‌های سایت 1 و 2
- `inventory_sync_attribute_mapping` - نقشه‌برداری ویژگی‌های سایت 1 و 2
- `inventory_sync_products_transferred` - ثبت محصولات منتقل‌شده

#### 4. **تب جدید: محصولات منتقل‌شده** ✅
- مشاهده تمام محصولاتی که موفقیت‌آمیز منتقل شده‌اند
- نمایش وضعیت انتقال هر محصول (موفق/ناموفق)
- نمایش وضعیت انتقال دسته‌بندی و ویژگی‌ها برای هر محصول
- جستجو در محصولات منتقل‌شده
- تاریخ انتقال برای هر محصول

#### 5. **بهبود متد Transfer** 🚀
- انتقال خودکار دسته‌بندی‌ها و ویژگی‌ها هنگام انتقال محصول
- انتقال صحیح متغیّرهای محصول متغیّر
- انتقال دقیق تصاویر محصول
- مدیریت بهتر خطاها و logging

### جزئیات پیاده‌سازی:

#### کلاس `Inventory_Sync_Category_Attribute_Sync`:
```php
// انتقال دسته‌بندی‌های محصول
$category_attr_sync = new Inventory_Sync_Category_Attribute_Sync($site1_api, $site2_api);
$category_map = $category_attr_sync->sync_product_categories($product['categories']);

// انتقال ویژگی‌های محصول
$attribute_map = $category_attr_sync->sync_product_attributes($product['attributes']);
```

#### بهبود API:
متد‌های جدید در `class-api.php`:
- `get_categories()` - دریافت دسته‌بندی‌ها
- `create_category()` - ایجاد دسته‌بندی جدید
- `get_attributes()` - دریافت ویژگی‌ها
- `create_attribute()` - ایجاد ویژگی جدید
- `get_attribute_terms()` - دریافت تکیه‌های ویژگی
- `create_attribute_term()` - ایجاد تکیه جدید

#### بهبور دیتابیس:
متد‌های جدید در `class-database.php`:
- `add_category_mapping()` - ثبت نقشه‌برداری دسته‌بندی
- `get_category_mapping()` - دریافت نقشه‌برداری دسته‌بندی
- `add_attribute_mapping()` - ثبت نقشه‌برداری ویژگی
- `get_attribute_mapping()` - دریافت نقشه‌برداری ویژگی
- `add_transferred_product()` - ثبت محصول منتقل‌شده
- `get_transferred_products()` - دریافت محصولات منتقل‌شده
- `is_product_transferred()` - بررسی انتقال محصول

### حل مشکلات:

1. **مشکل: دسته‌بندی موجود نیست** ✓
   - حل: انتقال خودکار دسته‌بندی‌ها و والدین آن

2. **مشکل: ویژگی‌ها موجود نیست** ✓
   - حل: انتقال خودکار ویژگی‌ها و تکیه‌های آن

3. **مشکل: نمی‌دانیم کدام محصول منتقل شده است** ✓
   - حل: تب جدید برای مشاهده محصولات منتقل‌شده

4. **مشکل: دوباره انتقال محصول** ✓
   - حل: ثبت محصولات منتقل‌شده برای جلوگیری از تکرار

### تغییرات فایل‌ها:

#### فایل‌های جدید:
- `includes/class-category-attribute-sync.php` - کلاس انتقال دسته‌بندی و ویژگی‌ها

#### فایل‌های تغییرتافته:
- `includes/class-database.php` - اضافه کردن جداول و متد‌های جدید
- `includes/class-api.php` - اضافه کردن متد‌های API برای دسته‌بندی و ویژگی‌ها
- `includes/class-sync-manager.php` - بهبود متد `transfer_product()`
- `includes/class-admin.php` - اضافه کردن AJAX handler برای محصولات منتقل‌شده
- `includes/class-plugin.php` - اضافه کردن require برای کلاس جدید
- `admin/dashboard.php` - اضافه کردن تب جدید برای محصولات منتقل‌شده
- `assets/js/admin.js` - اضافه کردن توابع برای تب جدید
- `assets/css/admin.css` - اضافه کردن استایل‌های جدید

### نحوه استفاده:

1. **انتقال خودکار دسته‌بندی و ویژگی‌ها:**
   - انتقال محصول را کلیک کنید
   - سیستم خودکار دسته‌بندی‌ها و ویژگی‌ها را منتقل می‌کند
   - اگر موجود بود، استفاده می‌شود؛ اگر نبود، ایجاد می‌شود

2. **مشاهده محصولات منتقل‌شده:**
   - به تب "✅ محصولات منتقل‌شده" برید
   - تمام محصولات منتقل‌شده را مشاهده کنید
   - وضعیت دسته‌بندی و ویژگی‌ها را بررسی کنید

3. **جستجو:**
   - نام محصول را در فیلد جستجو بنویسید
   - نتایج خودکار فیلتر می‌شوند

### مثال عملی:

**محصول: تیشرت خورشید**
- دسته‌بندی: تیشرت تک (سایت 1) → ایجاد شود در سایت 2 اگر موجود نبود
- دسته‌بندی والد: لباس زنانه → منتقل شود
- ویژگی‌ها:
  - رنگ: سبز، سفید، قرمز → ایجاد شوند اگر موجود نبودند
  - سایز: 35، 40، 45 → ایجاد شوند اگر موجود نبودند

**نتیجه:**
- محصول کامل منتقل می‌شود
- دسته‌بندی‌ها و ویژگی‌های آن نیز منتقل می‌شوند
- محصول در تب "محصولات منتقل‌شده" نمایش داده می‌شود با علامت ✓
