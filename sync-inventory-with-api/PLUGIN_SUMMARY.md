## 🔄 جریان کار کامل:

### مرحله 1: نصب (1 دقیقه)
```
فولدر کپی → WordPress Dashboard → Activate
```

### مرحله 2: تنظیمات (5 دقیقه)
```
WooCommerce > REST API > Create Keys
       ↓
Copy Consumer Key + Secret
       ↓
Paste in Plugin Settings
       ↓
[🔗 Test Connection] → ✓
```

### مرحله 3: انتقال (حسب تعداد)
```
[📤 Transfer All Products]
       ↓
Progress Bar ████░░░░░░ 45%
       ↓
✓ 50 Products Transferred
```

### مرحله 4: مرتبط‌سازی (1 دقیقه)
```
Product Site 1 ↔ Product Site 2
       ↓
Click to Match
       ↓
✓ Linked
```

### مرحله 5: فعال‌سازی (30 ثانیه)
```
☑️ Enable Auto Sync
       ↓
[💾 Save]
       ↓
✓ From Now: Automatic
```
